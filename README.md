<h1 align="center"> 🚜 Catálogo de Tratores </h1>

## 🌾 Sobre o Projeto

Projeto desenvolvido para a disciplina de **Programação Web**, ministrada pelo professor **Deivison**.

A ideia para este catálogo surgiu após horas de gameplay nos simuladores **Farming Simulator 2015 e 2019**. O interesse gerado por tratores, colheitadeiras e implementos agrícolas inspirou a criação desta página web, que funciona como um mostruário interativo de maquinário pesado.

## ✨ Funcionalidades

- 🎬 **Tela de Introdução:** Animação de transição suave para o catálogo principal.
- 📱 **Grid Responsivo:** Layout adaptável para exibição dos cards das máquinas.
- 🔍 **Modal Interativo:** Informações técnicas detalhadas (potência, motor, transmissão, peso e uso) acionadas ao clicar em um modelo.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação do conteúdo.
- **CSS3:** Estilização, layout (Grid) e animações.
- **JavaScript (Vanilla):** Manipulação de eventos e lógica de exibição do modal.
